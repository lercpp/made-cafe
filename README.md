#  Yami Maid Café

Современный веб-сайт ресторана, включающий функционал онлайн-меню, бронирования столиков, корзины заказов и личного кабинета пользователя.

---

##  О проекте

Данный проект представляет собой многостраничный сайт ресторана, разработанный на основе HTML.  
Он позволяет пользователям просматривать меню, добавлять блюда в корзину, оформлять бронирование столиков и управлять своим профилем.

Проект ориентирован на удобство пользователя и имитирует работу реального ресторанного сервиса.

---

##  Основной функционал

-  Просмотр меню ресторана  
-  Добавление блюд в корзину  
-  Оформление заказа  
-  Бронирование столиков  
-  Регистрация и авторизация пользователей  
-  Личный кабинет пользователя  
-  Навигация между страницами сайта  

---

##  Страницы проекта

- `main.html` — главная страница сайта  
- `meny.html` — меню ресторана  
- `korzina.html` — корзина заказов  
- `broni.html` — бронирование столиков  
- `profil.html` — вход  
- `reg.html` — регистрация нового пользователя  
- `ak.html` — профиль пользователя

---

##  Используемые технологии

- HTML5  
- CSS3  
- JavaScript
- Python(библиотека: FastAPI)
